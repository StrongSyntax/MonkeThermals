# MonkeThermals
A Gorilla Tag mod that gives you your own vertical air current! (boost controls and variable air current strength coming soon)

## Controls:
- Right Trigger:
> Starts Updraft

- Right Trigger Percentage (TO DO)
> Increases Updraft Strength

- A (TO DO)
> Boost
